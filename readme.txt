== Installation ==

1. Upload `wp-robax-software-for-wordpress` directory to Wordpress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'Robax' menu on the left.
4. Follow the instructions.
